Potential unicode characters detected in JSON. This could lead to basenames showing up weirdly in this list and in the game.
To resolve this issue, use the NomNom save editor (linked below) to copy the JSON.

It is recommended to use the NomNom save editor over the Java NMS Save Editor for copying the JSON out and into the save due to the Java Save Editor having problems with certain special characters which could result in weird behaviour.
